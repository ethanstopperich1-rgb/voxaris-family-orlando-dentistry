# V·OUTBOUND Reactivation Demo — Orlando Family Dentistry

Standalone interactive demo showcasing the Voxaris V·OUTBOUND reactivation engine for Orlando Family Dentistry.

## What This Covers

- **Lapsed Cleaning Patient** — Overdue hygiene reactivation call flow (warm greeting, need confirmation, preference capture, queue routing)
- **Invisalign Follow-Up** — Re-engagement of prior Invisalign interest leads (value framing, intent classification, warm tag and route)
- **Reactivation Flow Visual** — Five-stage pipeline: Segment → Prioritize → Outreach → Qualify → Convert
- **Benchmarks & Opportunity** — Industry-backed reactivation metrics (15–25% recovery, 35–40% return rate for 9–24 mo absent patients, ROI in 30 days)

## How to Use

Open `index.html` in any modern browser. No build tools or dependencies required.

1. Select a scenario (Lapsed Cleaning or Invisalign Follow-Up)
2. Press **Play Call** to watch the outbound conversation unfold
3. Observe the call flow progression panel updating in real time
4. Scroll down to see the full reactivation pipeline and opportunity metrics
