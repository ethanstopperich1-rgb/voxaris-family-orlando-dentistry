# Family Orlando Dentistry x Voxaris — Pitch Deck

16-slide + sources sales presentation generated with `python-pptx`.

## Quick Start

The `.pptx` file is pre-generated and ready to open:

```bash
open Voxaris_FamilyOrlandoDentistry_Demo_Deck.pptx
```

### Regenerate (optional)

```bash
pip install python-pptx
python3 generate_deck.py
```
